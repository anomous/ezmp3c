Tracks
This wizard page is used to select the audio tracks you want to extract and encode, edit tracks' information. The information is to be stored in output files as MP3 ID3 tags, Ogg comments, or WMA attributes.

Turn on check boxes of tracks to select the audio you want to extract and encode, and all other operations in this page can be accessed with the buttons on the bottom side or the popup menu of the track list. Each button has a tooltip hint, move mouse over it and you can get its function.

Refresh cd information.

Select all tracks. In the unregistered version, you can select odd tracks only.

Deselect all tracks.

Inverse the selection.

Configure freedb connection parameters (server, port and proxy).

Query information of your cd from freedb server.

Edit cd information, it is used to change information of all tracks.

Edit track information, it is used to change information of the selected track. You can double click one track to edit its information too.

Clear information, this will clear information of every track.
