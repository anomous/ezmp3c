Options
This wizard page is used to configure working parameters. It displays different content with different task. For example, with the task "CD -> MP3", it displays MP3 encoder parameters.

Output Folder: The folder to store output music files. You can use the button "..." to launch browse dialog or just type it in the edit box.

Filename: Format of output filenames. Support these macros.

<artist>: artist name of tracks
<album>: album label of tracks
<track>: track number
<title>: track title.

"\" can be used to create sub-directories.
You can select one of the pre-defined formats from the list or type your own format with customized strings and key words in the edit box directly.

Priority: Priority of working thread.

MP3 Encoder Parameters

  

Enable VBR: Use variable bitrate to encode audio data.

Write VBR Header: Write XING's VBR header in output MP3 files.

Quality: VBR Quality used to encode audio data with VBR enabled. 0 is the best while 9 is the worst.

Bitrate: Bitrate used to encode audio data.

Quality: Encode quality.

Audio Mode: Use stereo or mono channel mode.

Ogg Encoder Parameters

  

Bitrate: Bitrate used to encode audio data.

BBR: Use burstable bitrate to encode audio data.

VBR: Use variable bitrate to encode audio data.

Audio Mode: Use stereo or mono channel mode.

WMA Encoder Parameters

  

Bitrate: Bitrate used to encode audio data.

Audio Mode: Use stereo or mono channel mode.
